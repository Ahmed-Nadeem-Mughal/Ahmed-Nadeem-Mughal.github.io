Network Scanner & Banner Grabber

A custom reconnaissance tool built from raw sockets.

📁 Source Code 🖥 Live Demo
[ Screenshot / GIF of the project in action ]

What is this?

A multi‑threaded network scanner written in Python. It performs TCP port scanning, service version detection via banner grabbing, and outputs structured reports. I built it to understand the low‑level mechanics behind tools like Nmap, while keeping full control over the scan process.

Tech Stack

Python 3.11 Raw Sockets Threading Argparse JSON

Key Features

  • Multi‑threaded scanning (configurable thread count)
  • Banner grabbing for service identification
  • Export results to JSON or plain‑text
  • Stealth mode with random delays
  • Resumable scans (skip already scanned ports)

Code Sample

def grab_banner(ip, port):
    try:
        sock = socket.socket()
        sock.settimeout(2)
        sock.connect((ip, port))
        sock.send(b'HEAD / HTTP/1.0\r\n\r\n')
        banner = sock.recv(1024).decode().strip()
        return banner.split('\n')[0]
    except:
        return None

What I Learned

Building a scanner from scratch taught me how TCP handshakes really work, the importance of non‑blocking I/O, and the ethical considerations of network probing. It also became my go‑to lab for experimenting with concurrency patterns in Python.
